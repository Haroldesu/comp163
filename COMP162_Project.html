Introduction

This is the Las Vegas Strip dataset hostedd on the UCI Machine Learning Repository. The dataset can be found at this link: https://archive.ics.uci.edu/dataset/397/las+vegas+strip

The dataset includes 504 hotel reviews between January and August 2015. Some of the most important variables in this dataset are the user country, traveler type, hotel name, amenities of the hotel, the hotel star rating, and the score of the review.

In the code below, the dataset was cleaned correct formatting and number of stars for hotels that are rated 4.5 stars. I also created another column that determines if it is high quality or not depending on the hotel stars. In additions, I encoded the yes/no values for the hotel amenities into 0/1 to help with logistic regression later on.

In[2]:
import matplotlib.pyplot as plt
import numpy as np
import pandas as pd
import seaborn as sns
from sklearn import metrics, model_selection, linear_model, preprocessing

vegas = pd.read_csv("vegas.csv",delimiter = ";")
In[3]:
# Cleaning
vegas.loc[vegas['Hotel name'] == "The Cromwell", "Hotel stars"] = 4.5
vegas.loc[vegas['Hotel name'] == "Hilton Grand Vacations on the Boulevard", "Hotel stars"] = 3.5
vegas.loc[vegas['Hotel name'] == "Wyndham Grand Desert", "Hotel stars"] = 3.5
vegas.loc[vegas['Hotel name'] == "Marriott's Grand Chateau", "Hotel stars"] = 3.5
vegas["High Quality"] = vegas["Hotel stars"].map(lambda x: 1 if int(x) >= 4 else 0)
change_cols = ["Pool","Gym","Tennis court","Spa","Casino","Free internet"]
vegas[change_cols] = vegas[change_cols].map(lambda x: 1 if x == 'YES' else 0 if x == 'NO' else x)

Machine Learning

Logistic Regression

The first machine learning model that I wanted to test was to see if we can predict what makes a high quality hotel. Using the high quality variable I added in the beginning, I created a logistic regression model based on the amenities of the hotel. I trained the model outputed the confusion matrix and importance of each variable.

In[6]:
X = vegas.drop(["Hotel name","User continent","Member years","Review month","Review weekday","User country","Period of stay","Traveler type","High Quality","Hotel stars","Nr. reviews","Nr. hotel reviews","Helpful votes"],axis = 1)
X_train, X_test, y_train, y_test = model_selection.train_test_split(X,vegas["High Quality"], random_state = 333)
scaler = preprocessing.StandardScaler().fit(X_train)
X_train = pd.DataFrame(scaler.fit_transform(X_train), columns=X_train.columns, index=X_train.index)
reg = linear_model.LogisticRegression().fit(X_train,y_train)
y_pred = reg.predict(X_test)
labels = np.unique(y_test)
cm = metrics.confusion_matrix(y_test, y_pred, labels = np.unique(y_test))
display(pd.DataFrame(cm, index = labels, columns = labels))
display(metrics.accuracy_score(y_test, y_pred))
coef_df = pd.DataFrame(zip(reg.coef_[0], X_train.columns))
display(coef_df)

Linear Regression

Following a similar idea from as the logistic regression, I wanted to see if we can predict the number of hotel stars directly using same hotel amenities as the parameters for the linear regression model.

X = vegas.drop(["Hotel name","User continent","Member years","Review month","Review weekday","User country","Period of stay","Traveler type","High Quality","Hotel stars","Nr. reviews","Nr. hotel reviews","Helpful votes"],axis = 1)
